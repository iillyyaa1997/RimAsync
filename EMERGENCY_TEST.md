# ЭКСТРЕННЫЙ ТЕСТ: Полностью отключить все патчи

Создай файл `.env` в корне RimWorld с содержимым:
```
RIMASYNC_SKIP_HARMONY=1
```

Это отключит ВСЕ патчи RimAsync, мод будет загружаться, но ничего не делать.

Если после этого ошибки ИСЧЕЗНУТ - значит проблема действительно в патчах.
Если ошибки ОСТАНУТСЯ - значит проблема в другом месте (других модах, DLC, и т.д.)
